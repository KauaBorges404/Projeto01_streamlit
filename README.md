# Streamlit aplicativo interativo

Um aplicativo web simples e interativo construído com **Python** e **Streamlit** para demonstrar componentes de interface e visualização de dados.

## 🚀 Funcionalidades
- **Página Home:** Formulário com inputs de texto, seleção de curso, slider de notas e botões.
- **Página de Gráfico:** Geração dinâmica de gráficos de barras com valores aleatórios.
- **Navegação:** Menu lateral (sidebar) para alternar entre as páginas.

## 🛠️ Tecnologias
- [Python](https://www.python.org/)
- [Streamlit](https://streamlit.io/)
